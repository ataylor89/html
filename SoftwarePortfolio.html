Word Processor (Java)

Git: https://github.com/ataylor89/Java/blob/main/WordProcessor.java

Description: A graphical word processor that has many color schemes, several look and feels, creates files, saves files, opens files, sends email, compiles source code, links object code, and runs program binaries.

Screenshots: Writing a document Compiling a Java program Running a Java program Sending an email Copying to clipboard Saving a file Opening a file Selecting a color scheme Setting the foreground color Setting the background color Compiling an assembly language program Linking object code (part 1) Linking object code (part 2) Running a machine code program Running a Python program that draws a hexagon Running a Python program that draws a spiral of hexagons Compiling a C program with GCC Running a C program (part 1) Running a C program (part 2)

Dependencies: javax.mail.jar

Compile: javac -classpath javax.mail.jar WordProcessor.java

Run: java -classpath javax.mail.jar:. WordProcessor

Magic Squares (Java)

Git: https://github.com/ataylor89/Java/blob/main/MagicSquares.java

Description: Magic Squares is a game that lets you put the numbers one through nine in the squares of a 9x9 grid. You win the game when every row, column, and subgrid has the numbers one through nine.

Screenshots: Downloading a puzzle from the internet Solving a magic squares puzzle Checking if a solution is correct Choosing a color scheme (part 1) Choosing a color scheme (part 2) Choosing a color scheme (part 3) Choosing a color scheme (part 4)

Resources: Sample puzzle

Compile: javac MagicSquares.java

Run: java MagicSquares

Mathematical Art (Python)

Git: https://github.com/ataylor89/Python/blob/main/mathart.py

Description: Math Art is a Python program that creates mathematical art. Math Art draws polygons and spirals constructed from polygons.

Screenshots: Hexagon Hexagon Spiral Pentagon Pentagon Spiral Square Square Spiral Triangle Triangle Spiral Star of David Star Spiral

Run: python mathart.py <shape> <slength> <left|right> <x> <y>

Examples:

  1. python mathart.py hexagon 100 left
  2. python mathart.py star 100 right -100 0
  3. python mathart.py pentagonspiral 100 left
  4. python mathart.py starspiral 100 right

Sieve of Eratosthenes (NASM and C)

Git: https://github.com/ataylor89/nasm/blob/main/primes.asm

Git: https://github.com/ataylor89/C/blob/main/primes.c

Description: Eratosthenes is a Greek mathematician who created an algorithm for finding primes. The file primes.asm finds all prime numbers less than n. The file primes.c generates the first n primes.

Screenshots: Editing primes.asm Generating sieve of size 50 Editing and compiling primes.c Running primes.c Generating first hundred primes Generating first thousand primes Generating first million primes

Compile: nasm -fmacho64 primes.asm  or gcc primes.c -o primes

Link: ld -macosx_version_min 10.7 -lSystem primes.o -o primes

Run: ./primes or ./primes 100
